--------------------
Setup and Config
--------------------

config
	Configure your local git
	Read & Write
		Each level overrides the previous level.
			default 
				to .git/config eg C:\Users\Shaun\Documents\GitHub\BigFont\.git\config
			--global 
				to ~/.gitconfig eg C:\Users\Shaun\.gitconfig
			--system 
				to /etc/gitconfig eg C:\Program Files (x86)\Git\etc\gitconfig
	Your Identity
			user.name “John Doe”
			user.email johndoe@example.com
	Your Editor
			core.editor some-editor
	Your Diff Tool
			merge.tool some-merge-tool			
	Check Your Settings
			--list 
				Keys that are listed in multiple files are duplicated in this display.
			{key}
				Display what Git thinks is the value of that key.
	Examples
			$ git config --global user.name "Shaun Luttin”
			$ git config --global core.editor "'C:\Program Files (x86)\Notepad++\notepad++.exe' -multiInst -notabbar -nosession -noPlugin"
			$ git config --global merge.tool "'C:\Program Files (x86)\KDiff3\kdiff3.exe'"
			$ git config --list
		
help

Getting and Creating Projects
--------------------

init
	Create a local repository out of a local working directory
	Examples
			$ git init
clone
	Start tracking a existing remote repository
			[url]
			targetdir
	Examples
			$ git clone [git@github.com:shaunluttin/PracticeGit.git] practice
			$ git clone [https://github.com/shaunluttin/PracticeGit.git] practicegit
